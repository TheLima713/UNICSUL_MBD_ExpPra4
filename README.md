# UNICSUL_MBD_ExpPra4
Modelagem de Banco de Dados fictício para um sistema de registro de empréstimos de livros para a biblioteca LivroLivre
Nesse repositório estão os scripts que seriam utilizados no dia a dia pelos voluntários (operadores da biblioteca), contendo:
- Criação de tabelas
- Inserção de registros
- Consultas base
- Scripts de atualização
- Remoção de registros do sistema

Para implementar esse modelo relacional, foi utilizado um banco MySQL, e os scripts create_tables.sql e insert_tables.sql são executados uma vez apenas. Após isso, é possível utilizar os scripts selects.sql, updates.sql e deletes.sql para as funções diárias da biblioteca
